Key Features

Built for agent developers who need confidence in their A2A implementation.

🔐 Signature Verification

Verify JWS signatures on Agent Cards via JKU public key resolution. Supports P1363 and DER formats.

📊 Diff Reports

certify --compare baseline.json — detect regressions and improvements between certification runs.

🔒 mTLS Support

Client certificate authentication on discover, send, and certify with --mtls-cert and --mtls-key.

🧩 Extensions Framework

Test timestamp, traceability, and secure-passport extensions with 8 dedicated compliance tests.

🌐 Web Dashboard

Full browser GUI with interop testing, benchmarking, report history, and quick-action buttons.

⚙️ CI/CD Ready

GitHub Action included. Badge generation, HTML reports, and JSON output for pipeline integration.

Quick Start

From zero to certified in under a minute.

Terminal
# Install
$ npm install -g a2a-overture

# Start a mock agent
$ overture mock

# Discover the agent
$ overture discover http://localhost:3000

# Send a message
$ overture send http://localhost:3000 "Hello!"

# Run compliance certification
$ overture certify http://localhost:3000

# Test multi-agent interop
$ overture interop http://localhost:3000 http://localhost:3001

# Benchmark performance
$ overture bench http://localhost:3000 --concurrency 10

# Scaffold a new agent
$ overture init --sdk python --name my-agent

# Launch the Web UI
$ overture web
